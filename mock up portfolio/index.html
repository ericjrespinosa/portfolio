projects

form validator

This is the standard form validator that can be used on any web page to keep track of emails, aid users in creating website user accounts, and track data such as email, name, address, and phone number.

typing speed application

The typing speed game allows users to challenge their typing abilities with a race against the clock game. Every time the user has a valid input, a new word appears with a little more time. If the user enters a wrong input, time is taken away. There is an easy, medium, and hard mode.

meal recipe application

This application fetches data from an API and allows users to input keywords and find recipes with that keyword.

controlled breathing application

Controlled breathing allows a person in stressful situations to calm down, relax, and asses their environments. This application seeks to aid users in timing their breaths and giving them prompts to breath in and breath out.
